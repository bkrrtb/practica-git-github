## Conexión SSH realizada

 La conexión SSH ha sido configurada exitosamente en este equipo.  
A partir de ahora, los comandos `git push` y `git pull` se ejecutan sin necesidad de ingresar contraseña ni token.

Pasos completados:
- Generación del par de claves SSH (`id_rsa` e `id_rsa.pub`).
- Registro de la clave pública en GitHub.
- Cambio de la URL remota de HTTPS a SSH.
- Prueba de conexión con `ssh -T git@github.com`.
